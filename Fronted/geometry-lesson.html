Geometry Notes

Welcome to Geometry Notes! This section summarizes the key concepts from the Geometry module in your Mathematics course.

  • Points, Lines, and Planes:

    1. Points

    A point in space is indicated by a set of three coordinates,.

    • Notation: \(A(x, y, z)\)
    • No size or shape — simply a location.
    Example: Represent the point \(P\) that is 3 units right along the x‑axis, 2 up along the y‑axis, and 5 forward along the z‑axis. → \(P(3, 2, 5)\)

    2. Lines

    A line in 3D is defined by:

    • a fixed point \(P_0(x_0, y_0, z_0)\);
    • a direction vector \(\vec{d} = \langle a, b, c \rangle\).

    Parametric equations express any point on the line as:

    x = x₀ + at  
y = y₀ + bt  
z = z₀ + ct

    (Here, \(t\) is the parameter.)

    Example: Line through \(P_0(1, 0, -2)\) with direction \(\vec{d} = \langle 2, 3, 1 \rangle\). Equations: x = 1 + 2t, y = 0 + 3t, z = –2 + t

    3. Planes

    A plane is specified by:

    • a point \(P_0(x_0, y_0, z_0)\) on the plane;
    • a normal vector \(\vec{n} = \langle A, B, C \rangle\) perpendicular to it.

    Cartesian form:

    A(x – x₀) + B(y – y₀) + C(z – z₀) = 0
    Example: Plane through \(Q(2, –1, 3)\) with normal \(\vec{n} = \langle 1, 4, –2 \rangle\). ⇒ 1·(x–2) + 4·(y+1) – 2·(z–3) = 0 ⇒ x – 2 + 4y + 4 – 2z + 6 = 0 ⇒ x + 4y – 2z + 8 = 0

    Intersection of Line and Plane

    1. Plug parametric line equations into the plane’s Cartesian equation.
    2. Solve for \(t\), substitute back to find intersection point.
    Example: Find where the line x = 1 + 2t, y = 3t, z = –1 + t meets the plane x + y – z – 4 = 0. ⇒ Substitute: (1+2t) + (3t) – (–1 + t) – 4 = 0 → 1 + 2t + 3t + 1 – t – 4 = 0 → 4t – 2 = 0 → t = ½. Point: x = 1 + 1 = 2; y = 1.5; z = –1 + 0.5 = –0.5. So intersection at \( (2,\;1.5,\;-0.5) \).

    4. Additional Notes & Geometry Tips

    • Direction vector of line is parallel to it; can scale freely (e.g., \(\langle2,3,1\rangle\) same as \(\langle4,6,2\rangle\)).
    • Normal vector to plane defines its “orientation”; perpendicular to every direction within plane.
    • Parallelism: A line is parallel to a plane if its direction vector is perpendicular to the plane’s normal vector (i.e., \(\vec{d} \cdot \vec{n} = 0\)).
    • Containment vs Intersection: If substituting line into plane yields identity (0=0), the line lies in the plane; if it yields contradiction, the line is parallel and distinct.
    • Angle between: Angle between two lines via their direction vectors; between a line and plane via line’s direction vector and plane normal.

    5. Consolidated Summary Table

    ConceptDefinitionEquation Form
    Point Single location in 3D \(P(x, y, z)\)
    Line Set of points through \(P_0\), in direction \(\vec{d}\) \(\begin{cases}x=x_0+at\\y=y_0+bt\\z=z_0+ct\end{cases}\)
    Plane All points satisfying perpendicularity to normal vector \(A(x−x_0)+B(y−y_0)+C(z−z_0)=0\)

    6. Practice Tips

    • Always write down direction/normal vectors clearly.
    • Check parallelism via dot‑product = 0.
    • To find line–plane intersection, substitute parametrics into the plane equation.
    • Rearrange to standard/canonical forms for clarity and grading.

    — End of summary for Points, Lines, and Planes in 3D.

  • Angles:

    ✅ What is an Angle?

    • An angle is the figure formed by two rays (sides) sharing a common endpoint (vertex).
    • Measured in degrees (°), typically using a protractor.

    🧭 Types of Angles

    • Acute angle: between 0° and 90° (exclusive).
    • Right angle: exactly 90°.
    • Obtuse angle: between 90° and 180° (exclusive).
    • Straight angle: exactly 180°.
    • Reflex angle: between 180° and 360° (exclusive).
    • Full angle (perigon): exactly 360°.

    📏 Special Angle Relationships

    • Complementary angles: two angles that sum to 90°.
      e.g., 30° + 60° = 90°
    • Supplementary angles: two angles summing to 180°.
      e.g., 110° + 70° = 180°
    • Adjacent angles: share a common side and vertex, with non-overlapping interiors.
    • Vertically opposite angles: formed by two intersecting straight lines; vertical angles are equal.
      Lines AB and CD intersecting at O produce ∠AOC = ∠BOD

    🔄 Angles in Standard Position

    • An angle is in standard position if its vertex is at the origin and its initial side lies along the positive x-axis.
    • Coterminal angles share the same terminal side when placed in standard position:
      e.g., 30°, 390° and –330° are coterminal.
    • The smallest positive angle coterminal with a given angle θ can be found by: `θmod 360°` (if θ > 360°) or `(θ + 360°)mod 360°` (if θ < 0°).
      –45° → –45 + 360 = 315°

    🖊️ Corrected Examples & Notes

    • Measuring angles: Place the protractor's center over the vertex and align the base line with one ray. Read the scale at the other ray.
    • Example: If one ray points to 0° and the other to 120°, the angle measures 120° → an obtuse angle.
    • Vertically opposite: If two lines cross forming 120° at one angle, the opposite angle is also 120°.
    • Coterminal: The smallest positive angle coterminal with 810° → 810 mod 360 = 90° (right angle).
    • Complementary: One angle is 37°, its complement = 90 – 37 = 53°.

    Tip: Always specify “smallest positive rotation” when finding coterminal angles to avoid ambiguity.

  • Triangles:

    1. Triangle Essentials

    Definition: A triangle has three sides and three interior angles summing to 180°.

    Angle sum property: \(∠A + ∠B + ∠C = 180°\).

    Example: If ∠A=50°, ∠B=60°, find ∠C. Solution: 180 – (50+60) = 70°.

    Types by Angles

    • Acute: all angles < 90°.
    • Right: one angle = 90°.
    • Obtuse: one angle > 90°.

    Types by Sides

    • Equilateral: all sides equal; angles = 60°.
    • Isosceles: 2 equal sides; base angles equal.
    • Scalene: all sides different; all angles different.

    2. Right‑Angled Triangles & Pythagoras’ Theorem

    Definition: Right triangle has one 90° angle. Label hypotenuse as c, legs as a, b.

    Pythagoras’ Theorem: \(a^2 + b^2 = c^2\).

    Example 1: a=3, b=4 → c = √(9+16)=5.
    Example 2: c=13, b=5 → a = √(169−25)=12.

    Trigonometric Ratios

    For acute angle θ:

    • \(\sin θ = \frac{opposite}{hypotenuse}\)
    • \(\cos θ = \frac{adjacent}{hypotenuse}\)
    • \(\tan θ = \frac{opposite}{adjacent}\)
    Example: In a right triangle with angle A = 30° and hypotenuse = 10:
    \(\sin30° = 0.5\), so opposite = 5.
    \(\cos30° ≈ 0.866\), so adjacent ≈ 8.66.

    3. Similar Triangles & Applications

    Similar triangles: same shape, angles equal, sides proportional.

    If ΔABC ∼ ΔDEF, then:

    • \(AB/DE = BC/EF = AC/DF\)
    • Corresponding angles equal (Angles A = D, B = E, C = F).
    Corrected Example: ΔABC with sides 6, 8, 10; ΔDEF with 9, 12, 15. Ratio = 6/9 = 2/3.
    ✅ They are similar since 2/3 = 8/12 = 10/15.

    Thales’ Theorem (Optional)

    In a semicircle, angle on the rim = 90°. If diameter AB and C point on circle, then ∠ACB = 90°.

    Example: A circle has diameter AB=10. Point C lies on the circumference. Then AC² + BC² = AB² = 100.

    Real‑life uses

    • Surveying: find heights with trigonometry.
    • Construction: ensure right angles using 3‑4‑5 triangles.
    • Navigation: angle-of-elevation calculations.
  • Quadrilaterals and Polygons:

    1. Definitions & Key Terms

    • Polygon: A 2D closed figure made by ≥3 straight line segments (sides), meeting only at their endpoints (vertices).
    • Simple polygon: Edges only meet at endpoints (no crossing).
    • Convex vs Concave: In Convex polygons, no internal angle >180°; in Concave, at least one angle >180°.
    • Regular vs Irregular: Regular polygons have all sides and interior angles equal; irregular ones do not.
    • Diagonal: A segment joining two non-consecutive vertices.

    2. Angle-Side Relationships

    For any n-sided polygon:

    • Sum of interior angles = (n–2)×180°.
    • Each interior angle (regular polygon) = [(n−2)×180°]/n.
    • Sum of exterior angles (one per vertex) always equals 360°.
    Example 1: Find the sum of interior angles of a hexagon. n = 6 ⇒ (6–2)×180° = 720°.
    Example 2: Regular pentagon interior angle = [(5–2)×180]/5 = 540/5 = 108°.

    3. Types of Quadrilaterals

    All quadrilaterals are four-sided polygons with interior angles summing to 360°.

    3.1 Parallelogram

    • Opposite sides parallel and equal.
    • Opposite angles equal; adjacent angles sum to 180°.
    • Diagonals bisect each other.
    If in parallelogram ABCD, ∠A = 70°, then ∠C = 70°, ∠B and ∠D = 110° each.

    3.2 Rectangle

    • All angles are 90°.
    • Opposite sides equal and parallel.

    3.3 Rhombus

    • All sides are equal; opposite sides parallel.
    • Opposite angles equal; diagonals are perpendicular and bisect opposite angles.

    3.4 Square

    • Has properties of both rectangle and rhombus.
    • All sides equal, all angles 90°, diagonals bisect at 90°.

    3.5 Trapezoid (Trapezium)

    • Only two opposite sides are parallel.
    • Angles on the same leg are supplementary.

    4. Relationships among Quadrilaterals

    Hierarchy: square → rhombus → parallelogram → general quadrilateral.

    A square is a special rhombus and rectangle; not all rectangles are squares.

    Explain why a square is a rectangle but not vice versa: A square has four right angles → rectangle. But a rectangle can have unequal sides → not a square.

    5. Diagonals & Areas

    • Parallelogram area: base × height (A = b × h).
    • Rectangle: A = l × w.
    • Rhombus: A = (d₁ × d₂)/2.
    • Square: A = side² or (d₁ × d₂)/2.
    • Trapezoid: A = ½ × (sum of parallel sides) × height.
    Rhombus with diagonals 10 cm and 8 cm: Area = (10 × 8)/2 = 40 cm².

    6. Special Properties & Decision Tests

    Given coordinates or angle/sides, tests to identify quadrilateral:

    • Opposite side equality + parallelism → parallelogram.
    • All right angles + parallel opposite sides → rectangle.
    • All sides equal + parallel opposite sides → rhombus.
    • All sides and angles equal → square.
    • Exactly one pair of parallel sides → trapezoid.

    7. Polygons Beyond 4 Sides

    • Hexagon (6 sides): Sum = 720°; regular → 120° each.
    • Heptagon (7 sides):Sum = 900°; regular → ≈128.57° each.
    • General rule applies: (n−2)×180°, exterior sum = 360°.

    8. Important Corrections & Common Pitfalls

    • Interior angles sum: always (n–2)×180°, not n×180.
    • Square ≠ just "any quadrilateral with equal sides": must also have right angles.
    • Diagonals in parallelogram bisect each other but aren't necessarily equal.
    • Trapezoid mid-segment formula: connects legs at the midpoint: mid = (base₁ + base₂)/2.

    9. Combined Problems

    In parallelogram ABCD, diagonals AC and BD intersect at O. Prove AO = OC and BO = OD by the property of bisected diagonals.
    A regular octagon: n = 8 → sum interior = (8−2)×180 = 1080°; each angle = 1080/8 = 135°.

    11. Study Tips & Notes

    • Always start with: number of sides → sum of interior angles → possible shapes.
    • Use parallelism and side equality to classify quadrilaterals.
    • Practice diagonal intersection problems to master geometry proofs.
    • Memorize main area formulas—most numerical problems depend on them.
  • Circles:

    1. Basic Elements of a Circle

    • Centre (O): The fixed point from which all points on the circle are equidistant.
    • Radius (r): Distance from centre O to any point on the circle.
    • Diameter (d): A chord passing through O; d = 2r.
    • Chord: A line segment connecting two points on the circumference.
    • Secant: A line intersecting the circle at two points.
    • Tangent: A line touching the circle at exactly one point, ⟂ radius at the point of contact.
    • Arc: A portion of the circumference.
    • Sectors & Segments: Sector = region between two radii & an arc; Segment = region between chord & arc.
    Example: If a circle has radius r = 5 cm, then its diameter is d = 10 cm. A chord of length 8 cm is 3 cm from the centre (by perpendicular distance).

    2. Circle Theorems & Angle Facts

    1. Radius ⟂ Tangent: Radius to point of tangency is perpendicular to the tangent.
    2. Angles in a Semicircle: Angle in a semicircle is a right angle (90°).
    3. Angle Between Tangent and Chord: Equals the angle in the opposite arc.
    4. Angles in the Same Segment: Angles in the same segment are equal.
    5. Opposite Angles in Cyclic Quadrilateral: Sum to 180°.
    6. Angles Subtended at Centre vs. Circumference: Centre angle = 2 × corresponding circumference angle.
    Example: In circle with centre O, chord AB subtends angle AOB = 100° at the centre. The angle on the circumference, ACB, is = ½×100° = 50°.

    3. Equations of a Circle

    In the Cartesian plane, a circle with centre (h, k) and radius r is given by:
    (x − h)² + (y − k)² = r²
    If centred at the origin, it simplifies to x² + y² = r².

    Example: For circle x² + y² = 25, centre is (0, 0) and radius r = √25 = 5.

    4. Lengths & Areas

    • Circumference: C = 2πr (equivalently πd).
    • Area enclosed by circle: A = πr².
    • Arc length (θ in radians): L = rθ.
    • Sector area (θ in radians): A = ½r²θ.
    • Segment area: Area of sector minus area of triangular portion (½r² sin θ for central angle θ).
    Example: A 60° sector of a circle with radius 10 cm:
    θ = π/3 → arc length = 10×π/3 ≈ 10.472 cm; sector area = ½·100·π/3 ≈ 52.36 cm².

    5. Constructing & Using Equations

    To construct the equation of a circle from 3 non-collinear points, substitute their coordinates into (x − h)² + (y − k)² = r² and solve the system for h, k, r.

    Tip: Expand and compare to the general form x² + y² + Dx + Ey + F = 0, then complete the square to find centre (–D/2,–E/2) and radius √[(D/2)² + (E/2)² - F].

    6. Radicals & Accuracy

    When circles involve √‑values (e.g., r = √2), leave final answers in exact surd form unless a decimal approximation is requested. Always specify units clearly and include a diagram when possible.

    🔧 For more advanced geometry (e.g. intersections of circles, power of a point, radical axis), consult the original Core and Subsidiary books.

  • Perimeter, Area, and Volume:

    Page 1: Perimeter of Polygons

    Perimeter = total distance around a shape (sum of all side lengths).

    • Rectangle: P = 2 (l + w)
    • Square: P = 4 a (where a = side length)
    • Triangle: P = a + b + c
    Example: Rectangle with l=8 cm, w=5 cm → P=2(8+5)=26 cm.

    Page 2: Perimeter of Circle

    Circumference (perimeter) of a circle:

    • P = 2πr
    • P = πd
    Example: Circle r=7 cm → P=2×π×7≈43.98 cm.

    Page 3: Area of Polygons

    Area = amount of space inside a shape.

    • Rectangle: A = l × w
    • Square: A = a²
    • Triangle: A = ½ × b × h
    • Parallelogram: A = b × h
    Example: Right triangle with base=6 cm, height=8 cm → A=½×6×8=24 cm².

    Page 4: Area of Circles & Composite Figures

    Circle: A = πr²

    Composite shapes: Combine or subtract known areas.

    Example: Circle r=10 cm → A=π×100≈314.16 cm².

    Page 5: Surface Area of 3D Solids

    • Cuboid: SA = 2(lw + lh + wh)
    • Cube: SA = 6a²
    • Cylinder: SA = 2πr² + 2πr h
    Example: Cylinder r=3 cm, h=7 cm → SA = 2π×9 + 2π×3×7 ≈ 18π + 42π = 60π ≈ 188.5 cm².

    Page 6: Volume of Solids

    • Cuboid: V = l × w × h
    • Cube: V = a³
    • Cylinder: V = πr² h
    Example: Cuboid l=12 cm, w=5 cm, h=3 cm → V=12×5×3=180 cm³.

    Page 7: Application Tips & Common Errors

    • Always use correct units: Perimeter (cm, m), area (cm², m²), volume (cm³, m³).
    • Composite shapes: Split complex shapes into familiar shapes.
    • Recheck formulas: Use height ⟂ to base for triangles.
    • Pi (π): Use 3.14 or symbolically until final step.
    • Accuracy: Double-check arithmetic for surface-area vs volume mixups.
  • Coordinate Geometry:

    Coordinate Geometry – Key Concepts & Corrected Examples

    1. Cartesian Coordinates & Distance Formula

    A point is represented as (x, y) in the plane. The distance between points A(x₁, y₁) and B(x₂, y₂) is:

    d = √[(x₂ – x₁)² + (y₂ – y₁)²]

    Example: Find distance between A(–2, 1) and B(3, –4): d = √[(3 + 2)² + (–4 – 1)²] = √[25 + 25] = √50 = 5√2

    2. Midpoint Formula & Section Ratio

    The midpoint between A and B is:

    M = ((x₁ + x₂)/2, (y₁ + y₂)/2)

    For a point dividing the line AB in ratio m:n:

    P = ((n x₁ + m x₂)/(m + n), (n y₁ + m y₂)/(m + n))

    Example: Midpoint of A(1,2) and B(5,6) is M = (3, 4).

    3. Area of Triangle by Coordinates

    For A(x₁, y₁), B(x₂, y₂), C(x₃, y₃):

    Area = ½ |x₁(y₂ – y₃) + x₂(y₃ – y₁) + x₃(y₁ – y₂)|

    Example: A(–2,1), B(3,2), C(1,5): Area = ½ |(–2)(2–5) + 3(5–1) + 1(1–2)| = ½ |6 + 12 – 1| = ½ ×17 = 8.5

    4. Equation of a Straight Line

    Common forms:

    • Slope‐intercept: y = mx + c
    • Point‐slope: y – y₁ = m(x – x₁)
    • General: ax + by + c = 0

    The slope between A and B is:

    m = (y₂ – y₁)/(x₂ – x₁)

    Example: Line through (1,2) with slope 3: y – 2 = 3(x – 1) → y = 3x – 1.

    5. Perpendicular & Parallel Lines

    Two lines y = m₁x + c₁ and y = m₂x + c₂ are:

    • Parallel if m₁ = m₂
    • Perpendicular if m₁ × m₂ = –1

    6. Circle & Conic Sections (Brief)

    Circle: center (h, k), radius r:

    (x – h)² + (y – k)² = r²

    Example: Circle center (2, –1), r=3: (x – 2)² + (y + 1)² = 9.

    Parabola, ellipse, hyperbola: generally in second‑degree forms (e.g.

    Parabola with focus (p,0) and directrix x = –p: y² = 4px

    7. Intersection of Lines & Circles

    To find intersection(s), solve the two equations simultaneously:

    For example with two circles, subtract equations → linear equation → substitute back.

    8. Locus & Regions

    A locus is the set of points satisfying a geometric condition, like “distance from A twice that from B.” Often leads to circle, line, or parabola.

    Example: Locus of P such that PA = PB is the perpendicular bisector of AB.

    9. Quick Reference Formulas

    • Distance: d = √[(x₂–x₁)² + (y₂–y₁)²]
    • Midpoint: ((x₁+x₂)/2, (y₁+y₂)/2)
    • Triangle area: see section 3
    • Line slope: (y₂–y₁)/(x₂–x₁)
    • Slope‐intercept form: y = mx + c
    • Circle: (x–h)² + (y–k)² = r²

    🔍 Study Tips & Notes

    • Always check the slope form before selecting equation style.
    • Plot points to confirm sign or intercepts.
    • Use the determinant method (triangle area) to check collinearity.
    • Watch for special cases: vertical lines (undefined slope), horizontal lines (slope=0).

  • Transformations:

    Transformations – Summary

    1. Translation (Slide)

    Slides a figure by vector (h, k). It's an isometry (distance preserved, orientation same).

    Rule:

    • (x, y) → (x + h, y + k)
    Example: Translate ΔA(1, 3), B(4, 6) by (–2, +1):
    A′ = (1–2, 3+1) = (–1, 4); B′ = (2, 7).

    2. Reflection (Flip)

    Reflection overturns over a line—equidistant across it. Isometry but reverses orientation.

    Coordinate rules:

    • Over x-axis: (x, y) → (x, –y)
    • Over y-axis: (x, y) → (–x, y)
    • Over y = x: (x, y) → (y, x)
    • Through origin (180° flip): (x, y) → (–x, –y)
    Example: Reflect P(2, –5) over x‑axis → P′ = (2, 5).

    3. Rotation (Turn)

    Turns figure around a point (usually the origin). Preserves size/orientation only for 360°.

    About origin:

    • 90° CCW: (x, y) → (–y, x)
    • 180°: (x, y) → (–x, –y)
    • 270° CCW (or 90° CW): (x, y) → (y, –x)
    Example: Rotate Q(3, 4) 90° CCW → Q′ = (–4, 3).

    4. Dilation (Resize)

    Enlarges/shrinks figure by scale factor k from a center (often origin). Not isometric unless k=±1.

    Rule (from origin):

    • (x, y) → (k × x, k × y)
    Example: Dilate R(2, –3) by k = 3 → R′ = (6, –9).

    5. Combined & Algebraic Transformations

    Graphs of functions can be transformed using:

    y = a·f[b(x – h)] + k
    • h: right shift if +, left if –
    • k: up if +, down if –
    • a: vertical stretch (|a|>1) or shrink (0<|a|<1); flips if a < 0
    • b: horizontal shrink if |b|>1 or stretch if <1
    Example: Starting with f(x)=x²:
    y = –2·(x+3)² + 4 → shift left 3, stretch by 2, reflect over x‑axis, shift up 4.

    Key Notes

    • Translations, reflections, rotations are isometries (preserve distance).
    • Dilations preserve shape (angles) but not size.
    • Composition of transformations follows function order (rightmost first).
