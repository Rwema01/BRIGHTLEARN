Algebra Notes

Welcome to Algebra Notes! This section summarizes the key concepts from the Algebra module in your Mathematics course.

  • Expressions & Equations:

    1. Definitions & Key Concepts

    • Expression: A combination of numbers, variables, and operations—without an equal sign. Example: 3 x + 5, –2a² + 7b – 4 :contentReference[oaicite:1]{index=1}
    • Equation: A mathematical sentence with an equal sign, indicating two expressions are equal. Example: 3x + 5 = 11 or –2a² + 7b – 4 = 0
    • Variable: A symbol (usually a letter) representing an unknown value.
    • Coefficient: The numerical factor multiplied by a variable (e.g. in 5x, 5 is the coefficient).

    2. Operations & Properties

    • Order of Operations: Use PEMDAS (Parentheses, Exponents, Multiplication/Division, Addition/Subtraction).
    • Like Terms: Terms with the same variable and exponent. Example: 4x and –2x are like terms; 3a² and 5a² combine to 8a².
    • Distributive Property: a(b + c) = ab + ac. Example: 3(x + 4) = 3x + 12.

    3. Translating Between Words & Algebra

    • “The sum of twice a number and five” → 2x + 5.
    • “Three less than a number equals eleven” → x – 3 = 11.

    4. Evaluating Expressions

    Replace variables with given values and simplify:

    • If x = 2, evaluate 3x + 5 → 3·2 + 5 = 11.
    • If a = –1, b = 3, evaluate –2a² + 7b – 4 → –2·(–1)² + 21 – 4 = –2 + 21 – 4 = 15.

    5. Solving Equations

    Use inverse operations to isolate the variable:

    • One-step: Example: x + 4 = 9 → x = 9 – 4 → x = 5.
    • Multi-step: Example: 2x – 3 = 7 → add 3: 2x = 10 → divide by 2: x = 5.
    • Check: Plug solution back in: 2·5 – 3 = 10 – 3 = 7 ✓

    6. Equations with Variables on Both Sides

    • Example: 2x + 3 = x + 7 → subtract x: x + 3 = 7 → subtract 3: x = 4
    • Double-check: 2·4 + 3 = 8 + 3 = 11 and 4 + 7 = 11 ✓

    7. Common Errors & Tips

    • ▶️ Always apply operations to both sides of an equation.
    • ▶️ Be precise with signs: distributing ⟶ e.g. –2(x – 3) = –2x + 6, NOT –2x – 6.
    • ▶️ Combine like terms before solving.
    • ▶️ After finding the solution, always substitute back to verify.

    8. Example Problems With Notes

    1. Translate: “Five more than three times a number is 20” → 3x + 5 = 20. Solution: 3x = 15 → x = 5. Check: 3·5 + 5 = 20 ✓
    2. Solve: –4y + 2 = 10. → –4y = 8 → y = –2. Check: –4(–2) + 2 = 8 + 2 = 10 ✓
    3. Distributive with variables: 5(x – 2) = 3x + 4. → 5x – 10 = 3x + 4 → 5x – 3x = 14 → 2x = 14 → x = 7. Check: 5·5 – 10 = 15 = 3·7 + 4 = 25 (oops!) ⚠️ Mistake: plug in correctly: 5·(7–2)=25 and RHS=3·7+4=25 ✓

    9. Summary

    Expressive clarity (distinguishing expressions from equations), correct algebraic translation, and application of inverse and distributive properties are essential. After solving, substitution confirms accuracy. These techniques form a strong foundation for advanced algebra and real-world applications.:contentReference[oaicite:2]{index=2}

  • Variables & Constants:

    1. Definitions

    Variable: A symbol (usually a letter like x or y) that represents quantities which can change or vary. Variables are placeholders for unknown or varying values.

    Constant: A fixed value that does not change. In expressions and functions, constants are numbers like 3, −5, or 2.7.

    2. In Expressions

    Expressions combine variables and constants using operations like addition, subtraction, multiplication, and division.

    • Example: \( 2x + 3 \)
    • Here, x is a variable, because its value can change; 2 and 3 are constants.

    3. In Polynomials and Functions

    A polynomial is built from variables and constants using only addition, subtraction, multiplication, and non-negative integer exponents.

    • Example: \( f(x) = ax^2 + bx + c \) — Here a, b, and c are constants (with a ≠ 0), and x is a variable :contentReference[oaicite:1]{index=1}.
    • If an expression has no variables, it is itself a constant, e.g. 23 is a polynomial with constant terms only :contentReference[oaicite:2]{index=2}.

    4. Constant Function

    A constant function always returns the same value, regardless of input.

    • Example: \( f(x) = 5 \) — here 5 is a constant and x does not affect the result :contentReference[oaicite:3]{index=3}.

    5. Variables & Constants in Equations

    Equations often separate variables and constants to solve for unknowns.

    • Example: \( 3x + 5 = 20 \) — move constants (5, 20) to one side, variable term (3x) to the other, then solve: \[ 3x = 15 \implies x = 5 \] :contentReference[oaicite:4]{index=4}.

    6. Summary Table

    ConceptSymbol/ExampleDescription
    Variable x, y Represents unknown or changing quantity
    Constant 5, 2.7, a, b, c Fixed value that does not change
    Polynomial ax² + bx + c Combination of variables & constants under specific rules
    Constant Function f(x)=5 Function with only a constant term

    7. Notes & Tips

    1. Constants are often denoted by letters like a, b, c, but they represent specific fixed values.
    2. Variables are placeholders—their actual value is found by solving equations or evaluating functions.
    3. Any number without a variable is itself a polynomial of degree 0 (a constant).
    4. In equations, isolate variable terms on one side and constants on the other to find solutions.
  • Linear Equations: Linear Equations — Core Math Notes
    📐 1. Definition

    A linear equation in one variable x is an equation of the form:

    ax + b = 0

    where a and b are constants and a ≠ 0. The solution is:

    x = −b/a

    Example 1: Solve 5x − 20 = 0
    Solution:
    • Add 20: 5x = 20
    • Divide by 5: x = 4

    ✅ One-step addition/subtraction then division.

    2. Equations Requiring Rearrangement

    Even if the variable appears on both sides, you can rearrange:

    Example 2: Solve 3x + 5 = 2x − 7
    Solution:
    • Subtract 2x: x + 5 = −7
    • Subtract 5: x = −12

    🚨 Always move variables to one side first, then constants.

    3. Fractional Coefficients

    Linear equations may include fractions; multiply by the LCD to simplify:

    Example 3: Solve (1/2)x − 3 = (1/4)x + 1
    Solution:
    • LCD = 4 → multiply each term:
    2x − 12 = x + 4
    • Subtract x: x − 12 = 4
    • Add 12: x = 16

    🧮 Converting to integer coefficients simplifies arithmetic.

    4. Word Problem → Equation

    Translate statements into algebraic forms. Label the unknown, set up balance.

    Example 4: “Seven more than twice a number is 23. Find the number.”
    Model: Let x = the number.
    “Twice a number” → 2x, “seven more” → 2x + 7.
    Equation: 2x + 7 = 23
    Solve: 2x = 16 → x = 8.

    ✅ Always state clearly what each part represents.

    5. System of Two Linear Equations

    Two variables require two equations. Use substitution or elimination.

    Example 5:
    Solve:
    (1) x + y = 10
    (2) 2x − y = 3
    Solution (Elimination):
    Add (1) and (2): 3x = 13 → x ≈ 13/3
    Substitute into (1): 13/3 + y = 10 → y = 10 − 13/3 = 17/3

    🔍 Write answers as exact fractions when needed.

    ✅ Key Points Summary
    • Standard form: ax + b = 0
    • One-step: Isolate x by reversing operations.
    • Two-sided: Gather like terms before solving.
    • Fractions: Multiply by LCD to clear denominators.
    • Word problems: Define variable, translate precisely.
    • Systems: Use elimination or substitution; keep equations aligned.

    Tip: Plug your answer back into the original equation to check.

  • Exponents & Powers:

    Exponents & Powers

    Definition: If a is a real number and m is an integer, am is called a power or an exponential expression; a is the base, and m is the exponent or index. :contentReference[oaicite:1]{index=1}

    Basic Rules for Integral Exponents

    • Product of same base: am · an = am+n
    • Quotient of same base: am ÷ an = am–n, a ≠ 0
    • Power of a power: (am)n = am·n
    • Power of a product: (ab)n = an bn
    • Power of a quotient: (a/b)n = an/bn, b ≠ 0

    Important Special Cases

    • a0 = 1 (provided a ≠ 0)
    • a1 = a
    • a–n = 1 / an, a ≠ 0
    Note: A zero exponent gives 1. A negative exponent inverts the base’s power.

    Extension: Rational & Real Exponents

    If m and n are positive integers (with n ≠ 0), then
    am/n = (√na)m = √n(am), a ≥ 0
    More generally, ar is defined for any real exponent r by continuity and logarithms.

    Worked Examples (Corrected)

    1. Product: 32 × 34 = 36 = 729
    2. Quotient: 55 ÷ 52 = 53 = 125
    3. Power of a power: (23)2 = 26 = 64
    4. Negative exponent: 4–2 = 1 / 42 = 1/16
    5. Zero exponent: (–7)0 = 1
    6. Fractional exponent: 93/2 = (√9)3 = 33 = 27

    Why It Matters

    Exponents simplify expressions involving repeated multiplication and are foundational in topics like scientific notation, exponential growth/decay, indices in algebra, polynomials, and logarithms. :contentReference[oaicite:2]{index=2}

    Common Mistakes

    • Forgetting a0 = 1 (e.g., getting 00 wrong — it's undefined).
    • Applying the rules only for integers—forgetting how negative and fractional exponents work.
    • Misplacing parentheses: (–2)4 = 16 but –24 = –16.
  • Factoring:

    Factoring Polynomials

    1. Common Factor

    Example: Factorise 6x² + 9x → 3x(2x + 3).

    Note: Always extract the largest common monomial.

    2. Quadratic Trinomials

    Example: x² + 7x + 12
    Find m, n = 3, 4 → rewrite x² + 3x + 4x + 12 → (x+4)(x+3).

    Note: Confirm by expanding.

    3. Special Products

    Diff. of squares: x² – 9 = (x – 3)(x + 3)
    Perfect square: x² + 6x + 9 = (x + 3)².

    Note: Recognise these patterns for fast factoring.

    4. When a ≠ 1

    Example: 2x² + 5x + 3
    m, n: 2 & 3 → rewrite 2x² + 2x + 3x + 3 → (2x+3)(x+1).

    ✅ Checking

    Always re‑expand factors to verify they match the original expression.

  • Quadratic Equations:

    Quadratic Equations

    A quadratic equation is a polynomial equation of degree 2, of the form:

    ax2 + bx + c = 0, where a ≠ 0.

    Learning Objectives

    • Understand the meaning and structure of a quadratic equation.
    • Master four solution methods:
      1. Factorisation
      2. Completing the square
      3. Quadratic formula
      4. Graphical interpretation
    • Apply methods to solve real-life problems such as projectile motion and area optimisation

    1. Factorisation Method

    Rewrite ax² + bx + c as a product of two binomials:

    (dx + e)(fx + g) = 0

    The solutions are where each factor equals zero.

    Corrected Example

    Solve x² − 5x + 6 = 0.

    Factorisation: x² − 5x + 6 = (x − 2)(x − 3) = 0 → x = 2 or x = 3

    2. Completing the Square

    Transform the quadratic into a perfect square plus a constant:

    Steps:

    1. Divide by a (if ≠ 1).
    2. Move the constant term to the right side.
    3. Add/subtract (b/2a)² to both sides.
    4. Factor the left as a square expression.

    Corrected Example

    Solve x² − 4x + 1 = 0.

    Write: x² − 4x = −1
    Add (−2)² = 4 to both sides: x² − 4x + 4 = 3
    Left side becomes: (x − 2)² = 3
    → x − 2 = ±√3
    → x = 2 ± √3

    — preserve equality when moving terms :contentReference[oaicite:1]{index=1}.

    3. Quadratic Formula

    General solution formula:

    x = [−b ± √(b² − 4ac)] / (2a)

    Corrected Example

    Solve 2x² + 3x − 2 = 0.

    Compute discriminant: Δ = 3² − 4·2·(−2) = 9 + 16 = 25
    → x = [−3 ± 5] / 4
    So x = (−3 + 5)/4 = ½ or x = (−3 − 5)/4 = −2

    4. Graphical Method

    Plot y = ax² + bx + c and find its x‑intercepts (roots). Useful for visualising roots and checking nature of roots (real vs complex).

    5. Nature of Roots

    Determined by the discriminant (Δ = b² − 4ac):

    • Δ > 0: Two distinct real roots
    • Δ = 0: One repeated real root
    • Δ < 0: Two complex conjugate roots

    6. Applications in Context

    Quadratic equations model:
    • Projectile motion (e.g. ball thrown upwards)
    • Area problems (e.g. maximize area given perimeter)
    You can solve these either graphically or algebraically as in syllabus :contentReference[oaicite:2]{index=2}.

    7. Summary of Methods Comparison

    MethodWhen to UsePros & Cons
    Factorisation When roots are rational Quick, but not always possible
    Completing the square Works always Teaches structure, good for vertex form
    Quadratic formula Universal method Always works; formula memorisation required
    Graphical For visual insight & approx roots Less precise analytically

    8. Key Tips

    • Always check if you can factorise before using the formula.
    • Use completing-the-square to find vertex form y = a(x − h)² + k.
    • Verify your roots by back-substitution.
    • Interpret Δ to decide on real or complex roots.
  • Word Problems:

    1. Understand the Problem

    • Read the problem carefully and more than once.
    • Underline or highlight key information and the final question.
    • Rephrase the problem using your own words to confirm understanding.
    Example: Tess paints 2 boards in 4 min, Allie paints 3 in 2 min. Fence has 240 boards. How long will they take working together?
    Tess’s rate = 0.5 boards/min, Allie’s = 1.5 boards/min ⇒ Together = 2 boards/min
    Time = 240 ÷ 2 = 120 minutes = 2 hours

    2. Make a Plan and Set Up Equations

    • Define variables clearly (e.g. let x be the number of days, minutes, etc.).
    • Translate the words into mathematical expressions or equations.
    • Choose the correct method: proportions, linear equations, unit rates...
    Example: A crop increases daily. Day 1 = 2 kg, Day 2 = 4 kg.
    Let daily increase be k: 4 - 2 = k = 2 kg
    General formula: Yield = 2 + 2 × (day - 1)

    3. Solve and Check

    • Follow your plan and solve step-by-step.
    • Substitute back to verify if the answer is reasonable.
    • Write a sentence to summarize the answer with correct units.
    Example: John has 4 fewer candies than Bob; Bob has 18.
    John = 18 - 4 = 14. Together: 14 + 18 = 32 candies.

    4. Use Special Techniques

    A) Two-Step Problems

    • Perform operations in sequence (e.g., subtract then add).
    Example: Sarah had $15, spent $8, then received $5.
    Step 1: 15 - 8 = 7, Step 2: 7 + 5 = $12

    B) Ratio and Proportion

    • Add parts of the ratio to divide total appropriately.
    • Multiply each part with unit value.
    Example: Charlie and David share 40 sweets in 3:5 ratio.
    Total parts = 8, Unit = 40 ÷ 8 = 5
    Charlie = 3×5 = 15, David = 5×5 = 25

    C) Proportion with Cross-Multiplication

    • Set up ratios as fractions and solve using cross-multiplication.
    Example: 12 eggs cost $1.49. How much for 18?
    Set up: 12/1.49 = 18/x → x = (1.49×18)/12 = $2.24

    5. Final Check – Pólya's Four Steps

    1. Understand the problem.
    2. Devise a plan.
    3. Carry out the plan.
    4. Look back and review the result.

    Always label your answer with correct units and reflect if your result makes sense in real life.
