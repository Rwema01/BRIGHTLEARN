How to Navigate the Platform

BRIGHTLEARN provides a clean, user-friendly interface with useful features:

  • Dashboard

    View summary stats, current course progress, and important notifications.

  • Courses

    Browse and select interactive lessons for Math, English & History.

  • History

    Review completed activities and quiz scores with visual feedback.

  • Help

    Access documentation, FAQs, and support resources.

Navigate between sections easily using the sidebar menu.
